# Rosa's Café

A cozy café website built with [Nuxt 3](https://nuxt.com).

## How to run

```bash
npm install
npm run dev      # http://localhost:3000
```

## How to build for production

```bash
npm run build
npm run preview
```

## Project structure

```
app/        → pages, components, layouts
public/     → static assets (images, fonts)
```
